# Databricks-Certified-Generative-AI-Engineer-Associate-Resource

This repository contains the learning resources, courses, and videos I used to successfully pass the **Databricks Generative AI Engineer Associate** certification.

If you are preparing for the exam, these materials will help you build strong fundamentals and understand real-world implementation patterns.

---

### About the exam preparation

Unlike any tradition exam where you have a defined syllabus and limited set of books to read, certification exams like this one is different. The best strategy is to go through lots of questions (with correct answers of course). Try looking at the other options as well - meaning if 4 options are given and the correct answer is option 'A' , try to learn about whats given in Option 'B', 'C' and 'D' as well. This will help you understand the overall depth of the exam, not to forget it will make a better genAI engineer.  

---

### Prerequisites

You should have high level knowledge about the below topics atleast :
1. Prompting and its various techniques
2. RAG (Extraction, Chunking, Embedding etc)
3. Fine-tuning
4. Multi-agentic system
5. Basic python and popular libraries related to genai
6. Popular models including in-house model developed by Databricks DBRX
